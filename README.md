# Technologies used

1. Laravel 9.x
2. Sanctum package

# No frontend used!

Implementasi API ini sama sekali tidak memiliki frontend.
Proses testing API dilakukan menggunakan Postman.

# Installation

**Clone Repositori**

```bash
git clone https://github.com/malvinval/ads-be-3-assignment.git
```

**Pindah ke Direktori Projek**

```bash
cd ads-be-3-assignment
```

**Instal Dependensi**

```bash
composer install
```

**Konfigurasi Lingkungan**

```bash
cp .env.example .env
```

**Generate key**

```bash
php artisan key:generate
```

**Jalankan Migrasi**

```bash
php artisan migrate
```

**Jalankan Server Lokal**

```bash
php artisan serve
```
